# Variables (متغیرها)

---

# مفهوم

Variable یعنی یک مکان در حافظه که اطلاعات داخل آن ذخیره می‌شود.

می‌توان آن را مثل یک جعبه در نظر گرفت.

داخل این جعبه می‌توان عدد، متن و ... قرار داد.

---

# Syntax

```python
name = "Ali"
```

سمت چپ

نام متغیر

سمت راست

مقدار

---

# مثال

```python
name = "Hossein"
age = 25

print(name)
print(age)
```

خروجی

```
Hossein
25
```

---

# تغییر مقدار

```python
age = 25

age = 26

print(age)
```

خروجی

```
26
```

متغیر همیشه آخرین مقدار را نگه می‌دارد.

---

# قوانین نام گذاری

درست

```python
user_name = "Ali"

total_price = 100

age = 20
```

غلط

```python
1name = "Ali"

user-name = "Ali"

class = "test"
```
